二、css 3 选择器

Relationship Selectors 关系系选择器

<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>JS Bin</title>
</head>
<body>
  <div></div>
  <p>1</p>
  <p>2</p>
  <p>3</p>
</body>
</html>

E + F 下一个满足条件的兄弟元素节点

div + p {
  background-color:red;
}

E ~ F 下面所有满足条件的兄弟元素节点

div + p {
  background-color:red;
}

Attribute Selectors 属性元素选择器

E[attr~='val'] 选中带有attr属性，并且属性值是以 val 独立存在的元素。

<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>JS Bin</title>
</head>
<body>
 <div class='a'>1</div>
 <div class='aa abc'>2</div>
 <div class='a abc'>3</div>
</body>
</html>

div[class~='a']{
  background-color:red;
}

E[attr | ='val'] 选中带有attr属性，并且属性值以 val 开头，或者 val- 开头的元素。
