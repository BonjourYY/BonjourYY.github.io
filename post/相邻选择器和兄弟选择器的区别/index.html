相邻选择器和兄弟选择器的区别

· 2021-02-23 · # css # 选择器

相邻选择器

基本语法：A + B { }

理解：选中A元素后面的 与A元素同级的 一个 B元素

<!doctype html>
<html>
  <head>
    <title>相邻选择器/兄弟选择器</title>
    <meta charset="UTF-8" />
    <style>
      div + p {
        color:red; // 2 变红 3 不变红
      }
    </style>
  </head>
  <body>
    <div><p>1</p></div>
    <p>2</p>
    <p>3</p>
  </body>
</html>

兄弟选择器

基本语法：A ~ B { }

理解：选中A元素后面的 与A元素同级的 所有 B元素

<!doctype html>
<html>
  <head>
    <title>相邻选择器/兄弟选择器</title>
    <meta charset="UTF-8" />
    <style>
      div ~ p {
        color:red; // 2 和 3 都会变红
      }
    </style>
  </head>
  <body>
    <div><p>1</p></div>
    <p>2</p>
    <p>3</p>
  </body>
</html>
